# Smart Task Manager

A simple task management app built with React to help you organize daily tasks.

## Features

- ✅ Add, edit, and delete tasks
- 🔴 Set priority (High, Medium, Low)
- 📅 Add due dates
- 🔍 Search and filter tasks
- 💾 Auto-saves in browser

## Technologies Used

- React
- CSS


## How to Run Locally

```bash
git clone https://github.com/Zackbyte-ss/smart-task-manager.git
cd smart-task-manager
npm install
npm start
